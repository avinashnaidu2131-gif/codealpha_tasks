# Basic Network Sniffer

A beginner-friendly Python network sniffer built using Scapy.

## Features
- Captures live network packets
- Displays source IP and destination IP
- Shows protocol type (TCP, UDP, Other)
- Runs in real time

## Installation
```bash
pip install scapy
python sniffer.py
